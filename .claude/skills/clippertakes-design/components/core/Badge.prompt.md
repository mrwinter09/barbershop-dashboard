Compact label / tone pill for short metadata. For a story's lifecycle use StoryStatusTag; for a neighborhood use NeighborhoodTag.

```jsx
<Badge tone="petrol">New this week</Badge>
<Badge tone="accent">Story 24 just added</Badge>
<Badge tone="clay" dot>Entrepreneur</Badge>
<Badge tone="moss">Volunteer</Badge>
<Badge tone="outline">2026</Badge>
```

- **tone**: `neutral` · `petrol` · `accent` (Rotterdam yellow) · `clay` · `moss` · `outline`.
- `dot` adds a leading dot in the current color.
